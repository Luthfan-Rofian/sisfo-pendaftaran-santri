# 🧾 Sistem Informasi Pendaftaran Santri Baru (PSB)  
## Ponpes Darul Ulum Tlasih

Sistem Informasi Pendaftaran Santri Baru (PSB) untuk Pondok Pesantren Darul Ulum Tlasih.  
🚧 **Status: In Development** — Saat ini sedang dikembangkan untuk mendukung proses registrasi online dan manajemen data santri secara terintegrasi.

---

## 📌 Tentang Proyek

Aplikasi ini bertujuan untuk mempermudah proses pendaftaran santri baru secara digital, baik bagi calon wali santri maupun pihak administrasi pesantren.

Dengan sistem ini, proses pendaftaran menjadi:
- Lebih cepat
- Lebih efisien
- Minim kesalahan pencatatan manual

---

## ✨ Fitur Utama (Planned & Ongoing)

- 📝 **Formulir Pendaftaran Online**  
  Calon santri dapat mengisi data secara mandiri melalui web.

- 📂 **Manajemen Data Pendaftar**  
  Admin dapat mengelola, memvalidasi, dan menyimpan data pendaftar.

- 📄 **Upload Dokumen**  
  Mendukung pengunggahan berkas persyaratan.

- 🔍 **Verifikasi & Seleksi**  
  Proses validasi data calon santri oleh admin.

- 📊 **Dashboard Admin**  
  Monitoring jumlah pendaftar dan status seleksi.

- 📱 **Responsif**  
  Dapat diakses dengan baik melalui perangkat mobile maupun desktop.
